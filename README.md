# Ultimate Quiz Game

🌟 A simple Python quiz game with 10 general knowledge questions. Test your knowledge and have fun!

## Features

- 10 multiple-choice questions (A/B/C/D)
- Random question order each time
- Immediate feedback for correct/wrong answers
- Final score percentage displayed

## Installation

1. Make sure you have Python installed (Python 3.6+ recommended).  
2. Clone the repository:

```bash
git clone https://github.com/saraissa7/python-quiz-game.git
3. Navigate to the project folder:
cd python-quiz-game

How to Play 

Run the game in your terminal or command prompt:

python quiz_game.py

Type yes to start the game.

Answer each question by typing A, B, C, or D and press Enter.

Your score will be shown at the end.

Compatibility

Works in PyCharm, Command Prompt, and PowerShell on Windows.

Also works on Linux and Mac terminals.
